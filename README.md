# AISchool — AI 算法速成代码库

一天速成 10 个核心 AI / 数据科学 / 图论算法的 Python 实现。每个算法精简至 **20-35 行核心代码**，配有详细注释和可运行示例。

## 快速开始

```bash
# 依赖仅需 numpy
pip install numpy

# 运行任意算法示例
python algorithms/01_kmeans.py
```
