# 一天内快速掌握常见 AI/ML 算法 Python 核心实现

本文档面向**笔试/机试**中需要手写或口述核心逻辑的场合，配合仓库内 `examples/algorithms_reference.py` 中的可运行样例使用。

## 学习目标（一天内可达成）

- 能**不看资料**写出每个算法的核心循环或关键公式对应的代码骨架。
- 能说明**时间复杂度**、**输入输出**、**1～2 个易错点**（如 K-Means 初值敏感、PageRank 阻尼因子）。
- 能在纸上或 IDE 里**15～30 分钟**内复现一个算法的「最小正确版本」。

## 一天时间分配（建议按块执行，而非严格整点）

| 时段建议 | 内容 | 产出 |
|---------|------|------|
| 块 1（约 2～3 小时） | 无监督/图：K-Means、PageRank、二分图 BFS、Kruskal | 能默写主循环与数据结构 |
| 块 2（约 2～3 小时） | 监督学习：KNN、决策树信息增益、线性回归梯度下降 | 能写距离、熵、梯度更新 |
| 块 3（约 2～3 小时） | 深度学习基础：卷积滑动窗口、Max Pooling | 能写四维/二维循环边界 |
| 块 4（约 1～2 小时） | 不平衡：SMOTE；总复习 + 闭卷默写 | 对照样例查漏补缺 |

若只有半天，优先：**K-Means、KNN、梯度下降、信息增益、卷积/池化**（考试高频）。

## 每个算法的「最小掌握清单」

### K-Means（簇心迭代）

- **核心**：分配样本到最近簇心 → 用簇内均值更新簇心，直到簇心变化小于阈值或达最大迭代。
- **复杂度**：O(迭代次数 × n × k × d)。
- **易错**：空簇（本样例用简单处理：跳过或保留旧心）；初始化影响结果。

### KNN（欧氏距离 + 投票）

- **核心**：对测试点算与所有训练点的欧氏距离，取 k 近邻，分类用多数票。
- **复杂度**：朴素实现 O(n×d)；可提 KD 树但未在样例中实现。
- **易错**：k 为偶数时平票需约定规则（如随机或取最近一类）。

### PageRank（幂迭代）

- **核心**：r = (1−α)/N + α·M·r，反复乘直到 ‖r−r'‖ 足够小；M 为列随机转移矩阵（或按出边归一化）。
- **易错**：悬挂节点（无出边）需单独处理；阻尼 α 常用 0.85。

### 决策树核心（信息增益）

- **核心**：熵 H(p) = −Σ p log₂ p；增益 = 父熵 − 加权子熵；选增益最大的划分。
- **样例**：单特征阈值划分 + 递归深度限制，便于对照教材。

### 卷积（滑动窗口）

- **核心**：输出位置 (i,j) 为核与输入对应 patch 的点积之和（可加 bias）。
- **样例**：单通道 2D valid 卷积，便于理解索引。

### 池化（Max Pooling）

- **核心**：每个池化窗口取最大值，步长与窗口大小决定输出尺寸。

### 二分图判断（BFS 染色）

- **核心**：BFS/DFS 给相邻点交替染色 0/1；若遇已染色且同色则非二分图。

### 最小生成树（Kruskal）

- **核心**：边按权排序，并查集判环，依次加入不形成环的边，直到 n−1 条边。

### 梯度下降（线性回归）

- **核心**：损失 MSE；梯度 ∂L/∂w = (2/m) Xᵀ(Xw−y)；w ← w − η·梯度。

### SMOTE（过采样）

- **核心**：对每个少数类样本，在同类 k 近邻中随机选一个，在连线上随机插值生成新样本。

## 推荐学习方法（高效）

1. **先读样例中的一个函数 + 注释**，再在纸上盖住代码写出函数签名和主循环。
2. **对照运行**：`python examples/algorithms_reference.py` 查看自检输出。
3. **只记「骨架」**：变量名可简写，但循环边界与归一化步骤不能错。
4. **睡前再过一遍** 10 个名字与一行核心公式（记忆巩固）。
