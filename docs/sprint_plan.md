# 考前冲刺计划

> 适用于考前 1-3 天。根据剩余时间选择对应的冲刺方案。

---

## 方案 A：考前一天（8 小时完整版）

按照 [`study_guide.md`](study_guide.md) 的六个阶段完整执行。

**时间线：**
```
08:00-10:00  第一阶段：迭代收敛组（K-Means, PageRank, 梯度下降）
10:00-11:30  第二阶段：分类/决策组（KNN, 决策树）
11:30-12:30  第三阶段：矩阵/窗口组（卷积, 池化）
12:30-13:30  午休
13:30-15:00  第四阶段：图算法组（二分图, Kruskal）
15:00-16:00  第五阶段：SMOTE + 全部填空练习
16:00-17:30  第六阶段：模拟考试 + 查漏补缺
17:30-18:00  考前速查：过一遍 cheatsheet.md
```

**关键纪律：**
- 每学完一个算法立刻合上屏幕默写一遍
- 默写不出来的地方标记下来，作为重点复习对象
- 模拟考试严格计时，不看资料

---

## 方案 B：考前半天（4 小时精简版）

只看代码骨架和记忆卡片，跳过详细注释。

**时间线：**
```
第 1 小时：快速过 10 个算法的记忆卡片（study_guide.md 第三节）
第 2 小时：做填空练习题（exercises/01_fill_in_the_blank.py）
第 3 小时：做模拟考试（exercises/02_mock_exam.py）
第 4 小时：针对错题回看对应算法代码 + 再次默写
```

---

## 方案 C：考前 1 小时（紧急速查版）

1. 快速过 [`cheatsheet.md`](cheatsheet.md)（15 分钟）
2. 默写自己最不熟的 3 个算法（30 分钟）
3. 再过一遍公式表和易错点（15 分钟）

## 薄弱点加强策略

### 如果「记不住代码」
→ 抄写 3 遍，每次不看上一遍的结果

### 如果「理解不了原理」
→ 运行代码的"理解辅助"输出，跟着数据走一遍

### 如果「公式记混」
→ 把所有公式写在一张纸上，找出相似点和区别点

### 如果「手算出错」
→ 用最简单的数据（2x2 矩阵、3 个样本）反复练
